# Smart Light IoT Project

## Description
This project uses ESP32, PIR sensor and LDR sensor to automatically control lights.

## Components Used
- ESP32 Development Board
- PIR Motion Sensor
- LDR Sensor
- Relay Module
- Breadboard
- Jumper Wires
- LED / Light
- Power Supply
- WiFi

## Software Used
- Arduino IDE
- VS Code
- Python (Streamlit)
- GitHub
